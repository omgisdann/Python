🐍 **Meus Projetos em Python**

Este repositório contém projetos desenvolvidos para praticar lógica de programação em Python, com foco em aprendizado e evolução contínua.

---

## 📂 Projetos

### 📒 Agenda de Contatos

Sistema em terminal com funcionalidades de:

* Adicionar contatos
* Listar contatos
* Buscar por email
* Atualizar dados
* Remover contatos

---

### 🎯 Jogo de Adivinhação

Jogo onde o usuário tenta adivinhar um número aleatório:

* Número entre 1 e 100
* Dicas de maior ou menor
* Contador de tentativas

---

### 🎮 Gerador de Histórias Aleatórias

Programa interativo onde o usuário cria um personagem e escolhe caminhos dentro de uma história:

* Criação de personagem (nome, gênero e características)
* Escolha de caminhos (castelo ou natureza)
* Eventos diferentes baseados nas decisões do usuário
* Sistema de tentativas no desafio do castelo
* Possibilidade de jogar novamente

---

## 🚀 Tecnologias

* Python 3

---

## 📌 Objetivo

Este repositório foi criado para acompanhar minha evolução em programação, reunindo projetos práticos e aplicando conceitos fundamentais de Python.
