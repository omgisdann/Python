# 🐍 Meus Projetos em Python

Este repositório contém projetos simples desenvolvidos para praticar lógica de programação em Python.

---

## 📂 Projetos

### 📒 Agenda de Contatos
Sistema em terminal com funcionalidades de:
- Adicionar contatos
- Listar contatos
- Buscar por email
- Atualizar dados
- Remover contatos

### 🎯 Jogo de Adivinhação
Jogo onde o usuário tenta adivinhar um número aleatório:
- Número entre 1 e 100
- Dicas de maior ou menor
- Contador de tentativas

---

## 🚀 Tecnologias
- Python 3
